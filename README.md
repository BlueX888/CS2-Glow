# 🏴About
An extremely simple, external Counter-Strike 2 Glow ESP.

## 🔧Usage
1. Clone the repository
2. Update offsets: https://github.com/a2x/cs2-dumper/tree/main
3. Build in `Release | x64`
4. Enjoy!
